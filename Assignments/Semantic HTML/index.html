PARTICIPATION 10%
Regular participation and attendance is critical to successfully learning web design. You will also need to be in class to learn how to do weekly assignments. You will also want to participate in class discussions to help you answer questions and improve learning.
IN-CLASS ACTIVITIES 30%
An extension of the in class activity where you will create or edit a web page or discuss an existing web page based on concepts presented during the weeks lecture.
MODULES 40%
Modules are the rough equivalent of a midterm. There are four modules that address different web coding techniques. Each will count for 10% of your final grade. The first module is done individually and meant to demonstrate each students coding abilities. The remaining modules are completed as a group and must be collaborative and cooperatively complete.
  1. Three-page demonstration website
    Due: January 27, 2015
  2. HTML5 - Semantic, structural and navigational markup
    Due: February 17, 2015
  3. Responsive Design: CSS3 media queries, SVG and other mobile first techniques.
    Due: March 24, 2015
FINAL PROJECT 20%
This is a comprehensive project. You will be given specifications for a small web application that will require applying responsive design techniques and JavaScript interactive elements.
Due: April 28, 2015
